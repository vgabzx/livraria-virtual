# 📚 Livraria Virtual

## 📖 Visão Geral

Sistema de uma livraria virtual que gerencia desde a aquisição até a venda dos livros.

## 🚀 Funcionalidades

- Cadastro de compradores
- Login
- Catálogo de livros
- Carrinho de compras
- Pagamento com cartão
- Cálculo de frete
- Promoções
- Controle de estoque
- Relatórios
- Recomendações

## 🧑‍💻 Metodologia

- Histórias de Usuário
- Critérios de Aceitação
- Backlog do Produto
