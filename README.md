# 📚 Livraria Virtual

## 📖 Descrição

Sistema de uma livraria virtual que gerencia desde a aquisição até a venda de livros.

## Funcionalidades

- Cadastro de usuários
- Compra de livros
- Promoções
- Integração com editoras
- Controle de estoque
- Relatórios
- Recomendações

## Metodologia

- Histórias de Usuário
- Critérios de Aceitação
- Backlog do Produto
